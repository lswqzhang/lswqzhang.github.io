安装MySql5.7

发表于 2018-02-11 | 分类于 数据库 ， MySQL

MySql5.7安装过程：

  1. ==拷贝mysql压缩到local目录==

    1. cp /home/soft/MySQL-5.7.15-Linux-glibc2.5-x86_64.tar.gz /usr/local/

  2. ==进入local目录==

    1. cd /usr/local/

  3. ==解压mysql压缩包==

    1. tar -xzvf mysql-5.7.15-linux-glibc2.5-x86_64.tar.gz

  4. ==重命名mysql目录==

    1. mv mysql-5.7.15-linux-glibc2.5-x86_64 mysql

  5. ==添加一个mysql组==

    1. groupadd mysql

  6. ==添加一个用户==

    1. useradd -r -g mysql mysql

  7. ==把mysql目录授权 给mysql用户==

    1. chown -R mysql mysql

  8. ==创建mysql-data目录==

    1. mkdir /home/mysql-data

  9. ==把mysql-data目录授权给mysql组中的mysql用户==

    1. chown -R mysql:mysql /home/mysql-data

  10. ==chgrp命令可采用群组名称或群组识别码的方式改变文件或目录的所属群组==

    1.  chgrp -R mysql /home/mysql-data

  11. ==进入mysql目录==

    1.  cd /usr/local/mysql

  12. ==拷贝support-files目录下的my-default.cnf文件到当前目录的my.cnf文件==

    1.  cp support-files/my-default.cnf ./my.cnf

  13. ==编辑my.cnf文件==

    1.  vi /usr/local/mysql/my.cnf

##==basedir 为mysql 的路径==

basedir = /usr/local/mysql

##==datadir 为mysql的 data 包，里面存放着mysql自己的包==

datadir = /home/mysql-data

##==mysql端口==

port = 3306

##==服务器ID==

server-id = 100

##==socket文件==

socket = /tmp/mysql.sock

  1. ==拷贝my.cnf文件到etc目录下（若etc目录下有则替换）==

    1.  cp -fr  my.cnf  /etc/my.cnf

  2. ==编辑etc目录下的profile文件==

    1.  vi /etc/profile

##==在profile文件最后添加==

export PATH=/usr/local/mysql/bin:$PATH

  1. ==重新加载etc目录下的profile文件==
    1.  source /etc/profile
  2. ==拷贝==
    1.  cp -a ./support-files/mysql.server  /etc/init.d/mysqld
  3. ==初始化==
    1.  ./bin/mysqld --initialize --user=mysql --basedir=/usr/local/mysql/  --datadir=/home/mysql-data

此处需要注意记录生成的临时密码，方便后继登录

启动MySql

  1. ==启动Mysql==
    1. /etc/init.d/mysqld start
  2. ==查看mysql状态==
    1. /etc/init.d/mysqld status
  3. ==登录mysql==
    1. mysql -u root –p
  4. ==修改密码==
    1. set password=password(‘A123456’)
  5. ==远程登录==
    1. grant all privileges on *.* to root@’%’ identified by ‘A123456’;
  6. ==生效设置==
    1. flush privileges;
